# Nova-login

Modern Animated WordPress Login Experience

---

## Description

Nova Login is a modern WordPress login enhancement plugin designed to completely transform the default WordPress authentication screen into a polished and visually engaging experience.

The plugin replaces the standard WordPress login interface with a professionally styled layout featuring animated gradients, floating background effects, responsive form layouts, and improved typography. Built as a lightweight plug-and-play solution, Nova Login activates instantly without requiring additional setup or configuration.

Designed for creators, agencies, portfolios, membership websites, and custom dashboard systems, Nova Login improves the appearance of WordPress login pages while remaining fully compatible with existing themes and authentication systems.

---

## Features

### Modern Login Interface
Transforms the default WordPress login page into a clean, premium-style experience with modern spacing, styling, and UI enhancements.

### Animated Background Effects
Includes smooth animated visual elements and floating background effects to create a dynamic and engaging login screen.

### Responsive Design
Optimised for desktop, tablet, and mobile devices to ensure the login page remains fully accessible across all screen sizes.

### Professional Typography
Improves readability and presentation using refined typography and balanced interface spacing.

### Custom Form Styling
Restyles login fields, buttons, labels, and links while preserving native WordPress authentication functionality.

### Lightweight Performance
Uses lightweight assets and CSS-driven animations to maintain fast loading speeds without unnecessary overhead.

### Plug & Play
No complicated setup required. Install, activate, and Nova Login automatically applies the new login experience.

---

## How It Works

1. Install and activate the plugin.
2. Nova Login automatically hooks into the WordPress login page.
3. Custom styling and animations are injected dynamically.
4. The login experience is instantly transformed without affecting the rest of the website.

## Installation

1. Upload the plugin to /wp-content/plugins/
2. Activate the plugin through the WordPress Plugins screen
3. Visit /wp-login.php
4. Enjoy the new login experience
