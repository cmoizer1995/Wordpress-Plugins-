# CM SketchUp Portfolio Plugin

## Description
The CM SketchUp Portfolio Plugin is a custom WordPress plugin designed to showcase SketchUp and 3D model projects using Elementor. It allows users to manage construction, architectural, and design-based projects directly from the WordPress dashboard while displaying them in a clean, structured frontend layout.

The plugin supports embedded 3D viewers, downloadable model files, and detailed project descriptions, making it ideal for professional portfolios.

---

## Features

- 🧱 Custom SketchUp Post Type
- 🖥️ Embed Sketchfab or iframe 3D models
- 📥 Download Model button
- 🔗 View Model link support
- 📝 Full WYSIWYG description editor
- 📱 Fully responsive layout
- 🎨 Built for Elementor compatibility
- 🏗️ Structured for construction & architectural projects

---

## Installation

1. Download the plugin ZIP file  
2. Go to WordPress Dashboard → Plugins → Add New  
3. Click Upload Plugin  
4. Upload the ZIP file  
5. Click Install Now  
6. Activate the plugin  

---

## Usage

### Creating a Project

1. Go to Dashboard → SketchUp → Add New
2. Enter your Project Title
3. Fill in the following fields:

- Version (optional)  
- Status (Concept, In Progress, Stable, Latest)  
- Description (WYSIWYG editor)  
- View Model URL  
- Download URL  
- Sketchfab Embed (iframe code)

4. Click Publish

---

### Displaying the Project

- Open your page in Elementor
- Use your custom layout (or theme template)
- The plugin outputs:
  - Project title
  - Description
  - Embedded model
  - Download button

---

## Embed Support

You can embed models using:

### Sketchfab
Paste the iframe embed code:
html <iframe src="https://sketchfab.com/models/XXXX/embed"></iframe> 

### Other Sources
- Google Drive (public preview links)
- Self-hosted iframe embeds

---

## Button Behaviour

- Download Model → opens in same tab  
- External model links → open in new tab
