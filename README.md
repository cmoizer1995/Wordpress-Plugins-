# CM Docs Documentation System

A custom WordPress documentation and portfolio system built using Elementor and Advanced Custom Fields (ACF). This plugin transforms standard pages into structured, reusable documentation layouts with dynamic content sections.

---

## 🚀 Overview

- Custom Elementor widget for structured documentation output  
- ACF repeater integration for dynamic content sections  
- Responsive card-based layout  
- Divider-style section headings  
- Global settings page for display control  

---

## 🧠 How It Works

- Content is managed through ACF fields attached to posts or custom post types  
- Repeater fields allow multiple entries for sections such as Overview and How It Works  
- The Elementor widget dynamically renders all content on the front end  
- Each section is output independently to maintain stability  
- A WordPress settings page controls global display behaviour (e.g. title toggle)  

---

## ⚙️ Features

---

## 📦 Installation

1. Upload the plugin ZIP via WordPress:
   - Plugins → Add New → Upload Plugin  
2. Activate the plugin  
3. Ensure ACF (Advanced Custom Fields) is installed and active  
4. Add content using ACF fields  
5. Use the Elementor widget to display content  

---

## ⚙️ Settings

The plugin includes a dedicated settings page:

WordPress → Settings → CM Docs

From here you can:
- Enable or disable the plugin title globally  
- Control how the plugin integrates with your theme header  

---

## 🎯 Usage

1. Create or edit a post  
2. Fill in ACF fields:
   - Description  
   - Overview (repeater)  
   - How It Works (repeater)  
   - Tech Stack  
3. Open the page in Elementor  
4. Add the CM Docs Widget  
5. Style or adjust layout as needed
