This is a custom WordPress live blogging system designed to work with Elementor, allowing real-time content updates to be displayed dynamically on the frontend.

It provides a structured backend interface using ACF repeater fields, so you can quickly add live updates while keeping everything organised. On the frontend, it renders those updates in a clean, card-based layout, giving it that modern live blog feel.

The system is built to integrate seamlessly with Elementor templates, so it doesn’t interfere with your design — it just slots straight in and works alongside your existing layout.

⸻

Overview

At its core, the system acts as a live blog engine, giving you a dedicated interface in WordPress to create and manage updates during live events.

It uses ACF repeater fields to structure each update — things like event title, time, user, and content — so everything stays consistent and easy to manage.

On the frontend, updates are displayed in real time, with automatic refresh and update detection when live mode is enabled. This means users always see the latest information without needing to reload the page themselves.

Everything is presented in a clean, card-based layout, which improves readability and mirrors the kind of layouts you’d see on professional news or sports sites.

It’s also fully compatible with Elementor, meaning you keep full control over your page structure, hero sections, and styling.

And importantly, it’s designed as a plug-and-play system. Once the plugin is installed and ACF is active, everything is set up automatically — no manual configuration needed. You can start creating live blog posts straight away.

⸻

How It Works

Updates are created directly in the WordPress dashboard using a repeater-based interface, allowing you to add multiple entries within a single post quickly and efficiently.

Each update is automatically given a timestamp when saved, so you always have accurate timing without needing to enter it manually.

When you switch a post to live mode, the frontend activates automatic refresh and update detection, giving users a real-time experience.

Updates are then displayed in reverse chronological order, so the newest content always appears first.

You can also manage updates easily — adding, editing, or deleting entries — with everything syncing properly through ACF to ensure the frontend always reflects the latest state.

⸻

Tech Stack

This system is built using WordPress, Advanced Custom Fields, Elementor, PHP, CSS, and JavaScript.
