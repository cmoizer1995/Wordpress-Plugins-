# Sidebar Menu System (Elementor Plugin)

A custom WordPress sidebar navigation system built for Elementor, designed to provide flexible menu placement and responsive layout control across desktop and mobile devices.

---

## 🚀 Overview

- Elementor-integrated sidebar menu widget  
- Flexible positioning system (left/right on desktop, top/bottom on mobile)  
- Responsive layout behaviour across all devices  
- Customisable spacing and alignment  
- Works with the WordPress menu system  

---

## 🧠 How It Works

- Navigation menus are created using the built-in WordPress menu system  
- The sidebar menu is added to pages via a custom Elementor widget  
- The widget allows dynamic positioning within the layout  
- Layout automatically adapts between desktop and mobile views  
- Styling and spacing are controlled through Elementor and CSS  

---

## ⚙️ Features

---

## 📦 Installation

1. Upload the plugin ZIP file  
2. Go to WordPress → Plugins → Add New → Upload Plugin  
3. Activate the plugin  
4. Create a menu via Appearance → Menus  
5. Open a page in Elementor  
6. Add the Sidebar Menu Widget  

---

## 🎯 Usage

- Select a menu within the widget  
- Choose layout position:
  - Left / Right (desktop)  
  - Top / Bottom (mobile)  
- Adjust spacing, alignment, and styling  
- Place the widget within your layout
