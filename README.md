CMUK Elementor Editor Cleaner v1.0.0

A safe Elementor white-label and editor cleanup plugin for WordPress by Connor Moizer.

CMUK Elementor Editor Cleaner allows you to visually clean the Elementor editor, hide widget categories, hide individual widgets, white-label the Elementor interface, and simplify the editing experience for clients, demo sites, portfolios, and multisite environments — without breaking Elementor itself.

⸻

Features

Safe Elementor Cleaning

* Does not disable Elementor
* Does not unregister Elementor core systems
* Works in a safe “panel-only” mode
* Elementor editor still loads normally

⸻

White Label Features

CMUK Editor Branding

* Custom editor brand name
* Custom editor header colour
* Custom white-label text colour
* Custom white-label text size
* Left / Center / Right alignment controls
* Custom admin footer text

Elementor Editor Cleanup

Hide or clean:

* Elementor logo
* Upgrade/pro buttons
* Finder button
* Notes button

* Help/learn links

Widget Category Controls

Hide Elementor categories individually:

* Basic
* General

* Site

* Theme

Widget Controls

Hide individual widgets from the Elementor panel:

* Core Elementor widgets
* Elementor Pro widgets
* Third-party widgets
* CMUK widgets
* Custom dynamically registered widgets

Widgets are hidden visually from the panel while keeping Elementor stable.

Safe Mode

CMUK Elementor Editor Cleaner is designed to avoid:

* Elementor loading crashes
* Broken editor panels
* Widget registration conflicts
* Theme Builder failures

The cleaner hides items visually after Elementor loads instead of disabling Elementor systems.

1. Upload the ZIP in WordPress:

2. Activate:
    * CMUK Elementor Editor Cleaner
3. Configure:
    * Settings → CMUK Elementor Cleaner
