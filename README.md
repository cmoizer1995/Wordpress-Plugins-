# CV-System
A custom WordPress CV builder designed to create and display structured CV content using Elementor.

The plugin provides a simple backend interface for managing personal and professional information, while delivering a clean, responsive frontend layout through a dedicated Elementor widget.

It is designed for portfolio websites, CV pages, and standalone personal sites where clear presentation of experience and skills is required.

⸻

Installation

⸻

Upload Plugin

Download the plugin ZIP file and upload it via the WordPress plugin installer.

⸻

Activate Plugin

Activate the plugin from the WordPress Plugins page.

⸻

Access CV Builder

A new CV section will appear in the WordPress dashboard.

⸻

Setup

⸻

Create a CV

Create a new CV entry and add your content using the built-in sections:

* Skills
* Work History
* Education
* Hobbies & Interests

⸻

Add Content

Each section allows multiple entries, enabling full control over how your CV is structured.

⸻

Publish

Once complete, publish the CV entry.

⸻

Displaying the CV

⸻

Open Elementor

Edit any page using Elementor.

⸻

Add Widget

Search for and add the following widget: CM CV

Configure Sections

Select which sections to display:

* Skills
* Work
* Education
* Hobbies

⸻

Toggle Dividers

Enable or disable section dividers depending on layout needs.

⸻

Content Structure

⸻

Work Entry

Each work item displays:

* Job Title and Location
* Company
* Date
* Description

⸻

Education Entry

Each education item displays:

* Course and Location
* Educational Group
* Date
* Description

⸻

Skills

Displayed as individual cards using the skill title.

⸻

Hobbies

Displayed using a rich text editor for flexible formatting.

⸻

Layout Behaviour

⸻

Typography

* Titles displayed in bold
* Supporting text displayed in standard weight
* Consistent font size across all content

⸻

Card Design

Content is displayed in a structured card layout for clarity and readability.

⸻

Responsive Display

The layout adapts across desktop and mobile devices.

⸻

Usage

⸻

The plugin can be used for:

* Personal portfolio websites
* Online CV pages
* About sections
* Standalone CV platforms

⸻

Notes

⸻

* The Elementor plugin is required for frontend display
* Empty fields are not rendered
* Sections can be reused across multiple pages
