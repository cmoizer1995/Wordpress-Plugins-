WordPress Menu ( scrolling )

An Elementor widget that transforms native WordPress menus into a smooth horizontally scrolling navigation system inspired by ITV-style mobile navigation layouts.

This version has been rebuilt to focus entirely on WordPress menus while keeping full Elementor styling controls, dropdown support, hidden scrollbars, and responsive mobile behaviour.

⸻

Features

* Native WordPress Menu support
* Automatic dropdown menus from child menu items
* Horizontal scrolling navigation
* Hidden scrollbar design
* Mobile swipe support
* Elementor live preview support
* Responsive alignment controls
* Responsive spacing controls
* Text size controls
* Unified text colour controls
* Hover overlay removal
* Lightweight front-end output
* Multi-level dropdown support
* Mobile-first responsive layout

⸻

Elementor Controls

Content

* Select WordPress Menu

Style

* Text Colour
* Text Size
* Space Between Items
* Alignment
* Dropdown Width
* Dropdown Background

⸻

Dropdown Support

Child items added inside WordPress Menus automatically become dropdown menus.

Supports:

* First-level dropdowns
* Multi-level dropdowns
* Hover and focus interaction
* Mobile-friendly behaviour

⸻

Mobile Features

* Smooth horizontal swipe scrolling
* Hidden mobile scrollbar
* No black scrollbar strip
* Preserves visible menu items during scrolling
* No hover flash or tap overlay
* Responsive across phones, tablets, and desktop

⸻

Elementor Preview Improvements

This version fixes:

* Alignment controls not updating in preview
* Space between controls not rendering live
* Text colour inconsistencies
* Front-end/editor mismatches

All controls now update live inside Elementor editor preview.

Installation

1. Upload the plugin ZIP in WordPress
2. Activate the plugin
3. Open Elementor
4. Drag in:
    WordPress Menu ( scrolling )
5. Select a WordPress menu
6. Style using Elementor controls

⸻
